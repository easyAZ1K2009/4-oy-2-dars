Type conversion

Type conversion-JavaScript dasturlash tilida saqlanayotgan malumot bir korinishi ikkinchi korinishga otkazish

Number

Number-primitiv o'rab tiruvchi obyektidan foydalanib , malumot turini son korinishiga otgazish imkoniyati mavjud. Agar berilgan malumot son korinishiga otgazishni imkoni bolmasa, NaN maxsus qiymati yuzaga keladi.

String-primitiv orab turuvchi obyektidan foydalanib, berilgan malumot turini string korinishiga otgazish imkoniyati mavjud

Type coersion

Type coersion-JavaScript dasturlash tilida operator turli xil qiymatlar ustida ish olib borayotganda ,ulardan birining avtomatik ravishda boshqa turga otishi.

JavaScript dasturlash tilida berilgan qiymatni Boolean korinishiga otgazilganda, ular Truthy yoki Falsy korinishiga keladi
